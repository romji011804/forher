# For Her - A Personal Gift Website

A heartfelt website with Bible verses and encouragement.

## Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Push this code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/your-repo-name.git
   git push -u origin main
   ```
3. Go to repository Settings → Pages
4. Under "Source", select "main" branch
5. Click Save
6. Your site will be live at `https://yourusername.github.io/your-repo-name`

## Alternative: Quick Deploy with Netlify

1. Go to [netlify.com](https://netlify.com)
2. Drag the `forher.html` file into the drop zone
3. Your site is instantly live!

## Rename the file (optional)

For GitHub Pages to work automatically, you might want to rename `forher.html` to `index.html`:
```bash
mv forher.html index.html
```
